	a active

ctrl+w arrow - activate buffer
:q - quits buffer
ctrl+w ctrl+o - Make current window the only one on the screen. all other windows are closed
:bd          - deletes the current buffer, error if there are unwritten changes
:bd!         - deletes the current buffer, no error if unwritten changes
:bufdo bd    - deletes all buffers, stops at first error (unwritten changes)
:bufdo! bd   - deletes all buffers except those with unwritten changes
:bufdo! bd!  - deletes all buffers, no error on any unwritten changes

:bw          - completely deletes the current buffer, error if there are unwritten changes (buffer wipeout)
:bw!         - completely deletes the current buffer, no error if unwritten changes
:bd	     - deletes buffer but it is left in storage
:bufdo bw    - completely deletes all buffers, stops at first error (unwritten changes)
:bufdo! bw   - completely deletes all buffers except those with unwritten changes
:bufdo! bw!  - completely deletes all buffers, no error on any unwritten changes

:set confirm - confirm changes (Yes, No, Cancel) instead of error

# Tabs

Sometimes it just seems like Vim is a sinking ship. To make a life raft of your
current session (buffers, tabs), you can enter command mode and:

  :mksession(!) mysession.vim

and load it with `vim -S mysession.vim` or in command mode:

  :source mysession.vim
